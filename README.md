# pebble-predictor
Predicts Stokes number and pebble flux in a smooth protoplanetary disk

The corresponding paper is Drążkowska, Stammler, and Birnstiel, submitted to A&A.

HOW TO USE THE CODE

See the `pebble_predictor_examples.ipynb` notebook.
